# CLAUDE.md ŌĆö Straddle CLI (`straddle`)

## What this is
A standalone Go CLI for Straddle's **Pay by Bank + Embed** APIs. Two surfaces from one
binary: a human CLI and an MCP server for agents. It also keeps a local SQLite mirror of
synced resources and adds settlement/return analytics the official `straddle` CLI lacks.

- Module: `straddle` (Go + Cobra). Binary name: `straddle`.
- Originally produced by the **CLI Printing Press** generator from the Straddle OpenAPI
  spec, then extended by hand. **This is now a standalone repo ŌĆö we do NOT regenerate it.**
  Make all changes here directly. (Files with a `// Generated by CLI Printing Press ŌĆö DO NOT
  EDIT` header came from the machine; since we don't regen, editing them is fine ŌĆö just keep
  new logic in its own files where practical.)

## Build / test / run
```bash
go build -o ./straddle ./cmd/straddle   # always build to ./, never /tmp
go test ./...                                          # must be green before done
go vet ./...
gofmt -w <changed files>                               # don't gofmt the whole tree blindly
./straddle --help
./straddle doctor                               # verify auth/connectivity
```
- Agent mode: `--agent` (= `--json --compact --no-input --no-color --yes`).
- Human color/rich: `--human-friendly` (off by default ŌĆö agent-safe).

## Layout
- `cmd/straddle/` ŌĆö entry point
- `internal/cli/` ŌĆö Cobra commands (one file per endpoint) + hand-authored novel commands (`straddle_*.go`)
- `internal/client/` ŌĆö HTTP client; applies `Config.Headers` to every request
- `internal/config/` ŌĆö `config.toml` load/save
- `internal/store/` ŌĆö SQLite store: typed tables (spec-derived columns) + generic `resources` fallback
- `internal/mcp/` ŌĆö MCP server (code-orchestration: a `search` + `execute` tool pair) and `cobratree/` (runtime mirror of CLI commands as MCP tools)
- `internal/straddleacct/` ŌĆö integration-type + `Straddle-Account-Id` gating (see below)
- `spec.json` ŌĆö the OpenAPI spec; **authoritative source for resource/response shapes**

## Key invariant: `Straddle-Account-Id` scoping (hand-authored ŌĆö don't break it)
`internal/straddleacct/` decides when the header is sent, by integration type:
- `setup --type account|saas|marketplace` ŌĆö set the type (sticky, in `platform.toml`).
- `use-account <id>` ŌĆö set the acting account (sticky); `use-account --clear` to unset.
- `--account <id>` ŌĆö per-call override on any command.
- Policy (47/23 split derived from the spec): charges/payouts create ŌåÆ **required** for `saas`+`marketplace`; customers/paykeys/bridge ŌåÆ scoped for `saas`, **no header** for `marketplace` (platform owns); account-management ops (accounts/orgs/representatives/linked-banks/onboarding) **never** use the header (they carry the account in the body/path).
- Both the CLI pre-run and the in-process MCP handler call `straddleacct`, so agents and humans are gated identically. Route any new account-scoped behavior through this package.

## Output rules
- **Agent/JSON output must stay byte-stable.** Never change it for cosmetics.
- Human mode today: opt-in color + `tabwriter` tables (`printAutoTable`); single-object `get`
  prints JSON/raw. There is **no framed "detail card" renderer yet** (a planned feature).
- Per-response field curation already exists: `compactFields` (`internal/cli/helpers.go`) and
  the typed store columns (`internal/store/store.go`). Per-resource shapes come from `spec.json`.

## How this was made
1. Spec resolved from `docs.straddle.com/api-reference/openapi.json`, enriched with
   `x-auth-env-vars: [STRADDLE_API_KEY]` and `x-mcp` (code-orchestration, hidden endpoint mirrors).
2. Generated by CLI Printing Press: ~71 endpoint commands + store + MCP + README/SKILL/AGENTS.
3. Hand-authored: 7 analytics features (`reconcile`, `pipeline`, `returns`, `review-queue`,
   `cashflow`, `expiring`, `sandbox`) + `sql`; then integration-type account scoping
   (`straddleacct`, `setup`, `use-account`); plus a fix to dependent typed-table population
   (`capability_requests` now populates via `parent_id`).
4. Verified: shipcheck 6/6, live sandbox dogfood, `go test ./...` green.
5. Moved out of the generator's scratch run dir into this standalone git repo (`~/clawd/cli`).
